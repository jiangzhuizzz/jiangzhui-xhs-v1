# 图片渲染优化设计方案

**日期**: 2026-03-19

## 目标

优化小红书图片渲染效果，提升配图视觉质量。

## 需求背景

当前渲染存在以下问题：
1. 主题CSS文件缺失，导致sketch等主题样式不生效
2. 配图卡片样式单一（仅白色圆角卡片）
3. 默认使用separator模式，内容分散在多张卡片

## 优化方案

### 1. 创建8个主题CSS文件

位置：`scripts/assets/themes/`

| 主题 | 风格 |
|------|------|
| default | 简约现代，白底卡片 |
| playful-geometric | 几何色块、渐变背景 |
| neo-brutalism | 粗边框、硬阴影、鲜明对比 |
| botanical | 植物自然风、柔和绿色 |
| professional | 商务蓝、专业简洁 |
| retro | 复古暖色调 |
| terminal | 暗黑终端风 |
| sketch | 手绘素描、纸质纹理 |

### 2. 增强卡片样式

- 卡片容器：每个主题独特边框、阴影、背景纹理
- Markdown内容：标题、列表、引用、代码块主题化
- 装饰元素：主题专属图形、图标

### 3. 调整默认分页模式

- 默认模式从 `separator` 改为 `auto-fit`
- 内容自动缩放填满一张卡片，减少碎片化

## 技术实现

1. 创建 `scripts/assets/themes/` 目录
2. 为每个主题创建 `.css` 文件，包含：
   - 卡片容器样式
   - Markdown元素样式
   - 主题变量定义
3. 修改 `render_xhs.py` 默认参数
4. 测试所有主题渲染效果
