# 小红书运营 - 飞书模板说明

> 基于此文档创建多维表格

---

## 一、选题库（多维表格）

创建路径：飞书 → 云空间 → 新建 → 多维表格 → 选题库

### 字段设置

| 字段名 | 类型 | 说明 |
|--------|------|------|
| 选题 | 文本 | 笔记主题/方向 |
| 类型 | 单选 | 情感共鸣/武汉生活/贷款干货/工具类 |
| 状态 | 单选 | 待写 / 已写 / 已发布 |
| 优先级 | 数字 | 1-5（5最高） |
| 创建时间 | 创建时间 | 自动记录 |

---

## 二、笔记库（多维表格）

创建路径：飞书 → 云空间 → 新建 → 多维表格 → 笔记库

### 字段设置

| 字段名 | 类型 | 说明 |
|--------|------|------|
| 标题 | 文本 | 笔记标题 |
| 选题来源 | 关联 | 关联选题库 |
| 正文 | 文本 | 笔记正文内容 |
| 话题标签 | 文本 | #武汉 #打工人 |
| 封面图 | 图片 | 封面图片 |
| 内容图 | 图片 | 内容图片（多张） |
| 状态 | 单选 | 草稿 / 待审核 / 已发布 |
| 笔记链接 | 链接 | 小红书笔记URL |
| 发布时间 | 日期 | 实际发布时间 |
| 曝光 | 数字 | 曝光量 |
| 点赞 | 数字 | 点赞数 |
| 收藏 | 数字 | 收藏数 |
| 评论 | 数字 | 评论数 |

---

## 三、获取表格 ID

1. 打开多维表格
2. 浏览器地址栏 URL 最后一段就是 table_id
   - 例如：`https://xxx.feishu.cn/bitable/xxxxx` → table_id = `xxxxx`

---

## 四、配置到 config.yaml

```yaml
feishu:
  app_id: "你的飞书应用ID"
  app_secret: "你的飞书应用密钥"
  table_id_topics: "选题库的table_id"
  table_id_notes: "笔记库的table_id"
```

---

## 五、飞书应用创建（如无可忽略）

1. 打开 https://open.feishu.cn/
2. 创建企业应用
3. 添加权限：
   - bitable:table:readonly
   - bitable:table:write
   - drive:file:upload
   - drive:file:download
4. 发布应用
5. 获取 app_id 和 app_secret
