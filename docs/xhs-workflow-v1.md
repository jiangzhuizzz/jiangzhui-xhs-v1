## 详细步骤

### 第1步：选题

从 `05-选题库-起号期.md` 取当日对应类型选题

---

### 第2步：生成文案

```bash
# 调用提示词模板，生成文案
```

**输出格式**：
```markdown
---
emoji: "🍜"
title: "主标题（≤20字）"
subtitle: "副标题"
type: "武汉生活类"
---

# 主标题

正文...

#标签1 #标签2 #标签3
```

---

### 第3步：检测文案

```bash
cd ~/.openclaw/workspace/skills/wuhan-xhs-ops/scripts
python3 check_xhs.py -t "文案内容"
```

不通过 → 自动修改 → 重新检测（最多3轮）

---

### 第4步：人工审核

- [ ] 图片视觉效果符合预期
- [ ] 文案口语化，读起来像真人
- [ ] 无敏感词、无硬广痕迹
- [ ] 开头有钩子，结尾有互动提问
- [ ] 确认发布时间段

---

### 第5步：渲染图片

```bash
python3 render_xhs.py <文件路径> --theme sketch --mode auto-split
```

输出：`picture/YYYY-MM-DD-标题/cover.png` + `card_1.png`

---

### 第6步：发布

```bash
python3 ~/scripts/upload_xhs.py \
  --images <封面图路径> <内容图路径> \
  --title "笔记标题" \
  --content "正文内容" \
  --tags "武汉,打工人" \
  --visibility 公开
```
